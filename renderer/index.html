功能介绍

  • 拖拽或点击添加文件，支持单个 / 批量 / 文件夹导入
  • 9 种命名规则：序号、查找替换、前缀后缀、删除文本、大小写转换、正则替换、日期命名、插入文本、扩展名修改
  • 文件排序（按名称、修改日期、创建日期、大小、类型）
  • 预览重命名结果，自动检测命名冲突
  • 多选文件，仅对选中的文件执行操作
  • 右键菜单：单个重命名 / 移除文件
  • 一键撤销操作
  • 导出重命名映射表（CSV 格式）

命名规则

使用 {n} 表示序号，例如：文档_{n}
支持 {n} 表示序号，例如：第{n}_
提示：支持 JavaScript 正则语法，可以使用捕获组 $1, $2
常用格式： YYYY-MM-DD YYYYMMDD YYYY-MM-DD_HHmm
使用 {date} 表示日期，例如：备份_{date}
提示：0 表示开头，负数表示从末尾开始计算
不包含点号，留空则移除扩展名
